    # Nav

    This HTML element represents a structure in a page intended for navigation links.

    ## Common uses

    - menus
    - tables of contents
    - indexes

    ## Attributes

    - the global attributes

    ## Usage notes

    This element is only intended for salient clusters of navigation links.

    Documents may have multiple nav elements to mark navigation for specific purposes.
    In these cases, aria-labelledby can be useful for accessibility.

    User agents might use this element to determine rendering priority.
