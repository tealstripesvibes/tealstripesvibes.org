    # Ol

    This HTML element is used to represent an ordered list of items, typically rendered as a numbered list.

    ## Attributes

    - the global attributes
    - reversed
    - start
    - type

    ## Usage notes

    These may be nested as deeply as desired in combination with ul elements.

    The ul element is generally synonymous with the menu element.

    Use the ol element only when the order of the items is important.
