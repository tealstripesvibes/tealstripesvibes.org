    # Article

   This HTML element represents a self-contained composition that is meant to be able to stand on its own.

    ## Common contents

    - a forum post
    - a magazine entry
    - a newspaper article
    - a blog entry
    - a product card
    - a comment
    - a widget
    - a control panel

    ## Usage notes

    A document may contain multiple articles.

    Each article should be identified, typically with a header containing a heading.

    Articles can be nested.

    The article publication date and time can be defined using the `datetime` attribute of a time element.
